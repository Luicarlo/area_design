OUR WORK PROCESS

Little phrase about the work process

  • Información del proyecto


    Nos reunimos con el cliente de manera presencial o mediante videoconferencia por Skype para comentar el proyecto..


    Solicitamos toda la información de la que se disponga que defina el proyecto, como planos CAD, materiales (solerias, sanitarios, carpinterías, etc), imágenes reales que se quieran integrar, número de imágenes 3D a realizar (diurnos, nocturnos, exteriores, interiores, integraciones de modelos 3D con imágenes reales, etc). También se acordarán los plazos de entrega de las imágenes 3D.

  • Modelado 3D


    Con la información de los planos del proyecto levantamos el volumen 3D y mandamos imágenes del modelo sin texturas al cliente para confirmar que el modelo 3D es correcto.

  • Ángulos de cámara


    Se mandan al cliente varios puntos de vista de cada una de las imágenes 3D que se quieren realizar para que el cliente pueda elegir el encuadre de la cámara.

  • Iluminación


    Se ilumina el modelo 3D según el tipo de imagen 3D que se quiera realizar (diurna, nocturna, etc).

  • Materiales


    Con la información de materiales del proyecto se texturiza el modelo 3D para poner los materiales

  • Imágenes 3D previas


    Se renderizan las imágenes en baja resolución y se envían al cliente para que este pueda ver el modelo iluminado y con materiales y así pueda hacer comentarios para darle el acabado deseado.

  • Renderizado en alta resolución y postproducción


    Se realizan los últimos ajustes con técnicas de postproducción para mejorar el acabado de las imágenes 3D en alta resolución. Posteriormente se realizan los últimos ajustes con técnicas de postproducción para mejorar el acabado de las imágenes 3D en alta resolución..

  • Envío de imágenes 3D


    Se mandan las imágenes en alta resolución con postproducción realizada para la aprobación del cliente.
